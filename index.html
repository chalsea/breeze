Breeze

A CMS built for business

Download this project as a .zip file Download this project as a tar.gz file

Breeze is designed as a platform to showcase your business.

Breeze provides intuitive front-end editing, and a clean separation of content strategy from visual design, so you can update your site with confidence. Breeze is built on industry-standard open-source components, so you're never locked in to a single supplier. When you're ready to to extend beyond a brochure, Breeze can extend to support eCommerce, blogging and custom application development.

Intuitive front-end editing

Breeze lets you see your pages as you add content. You can drag blocks of content to new positions, and modify content separately from presentation. It requires minimal learning to get started on the real work of communicating with customers.

Content strategy

Breeze provides structured content types that help you maintain consistency and plan what will appear on the site before you create it. This allows designers to work with confidence, with a common understanding of the site's purpose. There's built-in support for SEO features such as meta tags, redirects, and simple human-readable URLs. Breeze also helps you manage images, PDFs and other assets used on your site.

Design freedom

Breeze makes it easy to convert a static HTML design into a reusable layout with regions for placing content and dynamic elements. It uses real Ruby code instead of a proprietary template language, so there's no limit to what you can add to a page. You can make rapid adjustments to a live site with the built-in theme editor, or build offline and update your completed theme when it's ready. Breeze doesn't say no to designers.

Solid foundations, open for extension

Breeze is open source, and built with the industry-standard Ruby on Rails 3 platform. We've designed it throughout to make sure you retain ownership of all aspects of your site, and won't be locked in to a single supplier. Modules are available for eCommerce and blogging, and developers can extend Breeze with custom components to fit your particular business needs.

Getting started

  1. Create a Ruby on Rails 3.2 application: 
    rails new my_site
  2. Edit your Gemfile to include the line 
    gem "breeze", github: 'leftclick/breeze', branch: '1.0.x'
    and run bundle install
  3. Create a Mongoid configuration file: 
    rails generate mongoid:config
  4. Add the following to the bottom of config/routes.rb: 
    mount Breeze::Engine, :at => '/'
  5. Run the generator to create an intial page and admin user account: 
    rails generate breeze:install
  6. Remove the Rails default static page: 
    rm public/index.html
  7. Start your server, and visit localhost:3000. You should see the home page.
  8. Log in to the admin interface at localhost:3000/admin:
    Username: emergency@example.com
    Password: logmein

Contributing to Breeze

To contribute just fork the repository on Github and create a new branch

$ cd your_repo_root/breeze
$ git fetch origin
$ git checkout -b my-new-feature

make sure the tests are still passing 

bundle exec rake spec

then submit your pull request.
